This is an Obsidian plugin forked from Obsidian Nutstore Sync which was designed to sync an Obsidian vault with Nutstore WebDav.

This project aims to re-engineer it to become a general-purpose Obsidian syncing plugin not locked to Nutstore.

Detailed codebase map can be found in `codemap.md` in each directory. They can help you understand the structure of the project without having to read the code.

The engineering plan and codebase analysis can ve found in `PLAN.md`. Phase 1 is done.
